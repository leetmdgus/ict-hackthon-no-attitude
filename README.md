# 🌊 해키와 함께하는 즐거운 강릉&양양

> 2024년 강릉시 ICT 공공데이터 활용 공모전 출품작  
> AR 스탬프 투어와 AI 챗봇으로 즐기는 스마트 관광 도우미

## 📱 프로젝트 개요

강릉&양양의 관광 데이터를 활용하여 AR 기술과 AI 챗봇을 결합한 새로운 형태의 관광 가이드 애플리케이션입니다.

## 🛠 사용 기술
- Frontend: Flutter
- Backend: Django
- AR: Unity
- AI: Gemini API
- Infrastructure: AWS

## 💫 주요 기능
- AR 스탬프 투어 (80개 관광 스탬프)
- AI 챗봇 가이드
- 실시간 리더보드
- 관광지 정보 제공
